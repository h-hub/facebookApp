# Stock Market App

# About
This is a Webapp which was build using Angularjs and Nodejs and the requirments are mentioned in the Requirments.pdf. This applications is ready to deploy in a production environment.

# How to Run

Go to the mentioned folders run the commands.
  - runner/build file
```sh
$ npm install
$ bower install
give "5" as the answer
$ gulp release
$ gulp build
$ gulp publish
```
 - runner/server
```sh
$ npm install
```
- login/build_files
```sh
$ npm install
$ gulp publish
```
 - login/server
```sh
$ npm install
```
 - home/build_files
```sh
$ npm install
$ gulp publish
```
 - home/server
```sh
$ npm install
```
 - framework
```sh
$ npm install
```
 - api/build_files
```sh
$ npm install
$ gulp publish
```
 - api/server
```sh
$ npm install
```
 - runner/server
```sh
$ node app
```	

> After running the last command, this message will be shown in the terminal: 
```sh
$ app listening on port 3000!
```
