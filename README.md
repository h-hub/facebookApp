# workflowApp

# How to Run

Go to the mentioned folders tun the commands.
  - runner/build file
```sh
$ npm install
$ bower install
give "5" as the answer
$ gulp release
$ gulp build
$ gulp publish
```
 - runner/server
```sh
$ npm install
```
- login/build_files
```sh
$ npm install
$ gulp publish
```
 - login/server
```sh
$ npm install
```
 - home/build_files
```sh
$ npm install
$ gulp publish
```
 - home/server
```sh
$ npm install
```
 - framework
```sh
$ npm install
```
 - api/build_files
```sh
$ npm install
$ gulp publish
```
 - api/server
```sh
$ npm install
```
 - runner/server
```sh
$ node app
```
